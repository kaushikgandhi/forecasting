Using weather forecasts should be as easy as cooking.

We'll help you get the ingredients, leaving you to decide: what's for dinner?

Download to PostGIS

Python

from forecasting import models

nam = models('nam')
nam.connect(database='weather', user='chef')
fields = ['tmp2m']
nam.transfer(fields)

Use the API (it's free!)

Python

import urllib2
import json

url = 'http://api.getforecasting.com/KEY/40n/105.3w/temperature'
response = urllib2.urlopen(url)
temperature = json.loads(response.read())
